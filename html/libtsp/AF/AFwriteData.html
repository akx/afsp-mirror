AFwriteData

Routine

int AFwriteData (AFILE *AFp, const float Fbuff[], int Nval)

Purpose

Write data to an audio file (float input values)

Description

This routine writes a specified number of samples to an audio file. The float input data is converted to the audio file data representation. The file data representation is set on opening the file with routine AFopenWrite. This routine writes data sequentially to the file. A warning message is printed if the input values exceed the dynamic range of the file data representation.

Parameters

<- int AFwriteData
Number of samples written
-> AFILE *AFp
Audio file pointer for an audio file opened by AFopenWrite
-> const float Fbuff[]
Array of floats with Nval samples
-> int Nval
Number of samples to be written
